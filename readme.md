# 量化流程 V2.0
## 数据校验
上送数据，校验数据是否**缺失**，**异常**，**重复**，**格式错误**等，排除**ST**、**小市值**、**停牌**等数据
## 持仓校验
打印关注名单，形如[code,type]，打印当前持仓，确认是否有更新，上送变化持仓，形如[code, B1, 止损价, 持仓天数]，[code, B2, 止损价, 持仓天数]，[code, B3, 止损价, 持仓天数]，[code, 异动/倍量柱, 止损价, 持仓天数]，保存当前持仓。判断当前持仓若为短线且三日横盘，则次日必须拍出或放飞，防止看短做长
## 基本规则校验
持仓是否大于**3**或**4**天，是否白线>黄线，是否为次日必须拍出类型，输出持股评分
## 止损校验
是否达到预设止损位，是否达到量化止损位（滴滴、放量阴线、破N型前低）➡️若是则次日必须拍出
## 止盈校验
是否达到量化止盈位（连续两根中长阳、连续上涨）➡️若是则次日必须放飞
## B1校验
（白线>黄线）B1校验：判断是否J值小于**13**➡️判断是否**缩量、一致、地量低价、长阴短柱、阳线反包、回踩日/黄/白线、击穿对手盘、SB1**➡️若是，给出买入信号、止损价（或止损类型，当根或前根最低）、盈亏比（前高价差）、买入价格
## B2校验
（白线>黄线）B2校验：判断是否**放量反包、J<80、当日涨幅>=4%、短或无上影线**➡️若是，给出买入信号、止损价（或止损类型）、盈亏比、买入价格
## B3校验
（白线>黄线）B3校验：前一日是B2➡️判断是否**非大阴线（完美是一致+缩半量以下**➡️若是，给出买入信号、止损价（或止损类型）、盈亏比、买入价格
## 单针校验
判断是否满足单针下20、单针下30，若是，给出买入信号、止损价（或止损类型）、盈亏比、买入价格，⚠️买入第二天不涨或未脱离成本区必须拍出
## 滴滴买回校验
（白线>黄线）滴滴重新买入校验：前**2或3**日满足滴滴卖出条件，今日**放量反包**下跌期间全部交易量➡️若是，给出买入信号、止损价（或止损类型）、盈亏比、买入价格
## 异动校验
（黄线>白线）判断是否满足异动（参考中信证券）、监控中信证券异动➡️若是，加入关注名单，形如[code,type]
## 倍量柱校验
（黄线>白线）判断是倍量以上柱➡️若是，加入关注名单，形如[code,type]
## 回撤系统
输入不同均线、日线，输出以此为基准B1买入S1卖出收益率，输入不同买入点、卖出点，输出收益率，输入不同因子，输出收益率
## 风控系统
单一持仓占比风险、整体持仓占比现金流风险、行业集中风险、最大回撤风险、活跃市值风险、个股单日成交额220亿风险
