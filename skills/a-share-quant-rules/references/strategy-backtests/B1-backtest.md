# B1 回测策略规范（可扩展版）

## 1. 策略信息

- 策略编号：`B1`
- 版本：`v0.1`
- 适用市场：A股（股票现货）

## 2. 基础交易约束（必须生效）

- 遵守 A 股交易基本规则：涨跌停、T+1、停牌不可交易，最小交易单位、交易费用与滑点。
- 股票池读取时排除：
  - `ST/*ST`
  - `科创板`
  - 其他需排除板块（预留）：`[TODO: 如北交所、次新股、长期停牌等]`

## 3. 数据异常处理

### 3.1 涨跌幅限制

为防止极端行情导致的数据异常，回测中对单日涨跌幅进行限制：

- **主板股票**（代码不以300/688开头）：单日涨跌幅限制在 **±10%**
- **创业板/科创板股票**（代码以300/688开头）：单日涨跌幅限制在 **±20%**

当计算出的价格超过涨跌停限制时，自动调整为涨跌停价格：

```
if 涨跌幅 > 涨跌幅上限:
    价格 = 前一日收盘价 × (1 + 涨跌幅上限)
elif 涨跌幅 < -涨跌幅上限:
    价格 = 前一日收盘价 × (1 - 涨跌幅上限)
```

### 3.2 其他数据异常

- 涨停无法买入：跳过信号
- 跌停无法卖出：顺延至下一交易日
- 一字板（开盘=最高=最低）：标记为异常数据
- 成交量为0或异常放大：标记为异常数据

## 4. 买入条件

同时满足以下条件触发买入信号：
1. `J < 13`
2. `趋势线 > 多空线`

执行说明（默认）：
- 若当日满足买入信号，以次日开盘价尝试买入。
- 若次日开盘不可成交（涨停/停牌/流动性不足），记录未成交原因。

## 5. 止盈卖出条件（按顺序执行）

### 5.1 条件列表

1. 当日收盘 `J > 100`，则以次日开盘价卖出。
2. 当日收盘价相对买入价涨幅达到 `8%`，则以次日开盘价卖出当前持仓 `20%`。
3. 当日收盘价大于短期趋势线，偏离幅度达到 `15%`，则以次日开盘价卖出当前持仓 `20%`。
4. 当日收盘价大于短期趋势线，偏离幅度达到 `20%`，则以次日开盘价卖出当前持仓 `20%`。
5. 当日收盘价大于短期趋势线，偏离幅度达到 `25%`，则以次日开盘价卖出当前持仓 `20%`。
6. 若按顺序执行至步骤 5，则剩余仓位锁仓；直到出现“连续两天收盘价小于前一天的最小值”，以次日开盘价全部卖出。
7. 每天可以触发多次止盈，可连续多天触发。

### 5.2 执行顺序规则

- 同一天按步骤 `1 -> 2 -> 3 -> 4 -> 5 -> 6` 评估。
- 当天若多个分级止盈阈值同时满足，可累计触发多个 `20%` 卖出动作（受剩余仓位约束）。
- 每个动作对应一笔“次日开盘”卖单，按顺序排队执行。

## 5. 止损卖出条件

### 5.1 条件列表

1. 收盘价小于 `min(买入当天K线最小值*0.97, 买入前一天K线最小值*0.97)`，卖出半仓。
2. 收盘价连续两天小于多空线，卖出半仓。
3. 收盘价连续三天小于前一天最小值，卖出半仓。
4. 每天只能触发一次止损，但可连续多天触发。

### 5.2 执行规则

- 当日若多个止损条件同时满足，仅执行优先级最高的一条（默认优先级：`1 > 2 > 3`）。
- 止损卖单以次日开盘价执行，并受涨跌停/停牌等约束。

## 6. 冲突与边界处理（默认实现）

- 同日同时满足止盈与止损时：
  - 默认先执行止盈序列，再评估止损（预留可配置项）。
- “卖出半仓”按当前可用持仓数量 `50%` 向下取整到最小交易单位。
- 当可卖持仓不足最小交易单位时，合并到下一次可执行卖出动作。

## 7. 回测输出要求

- 信号统计：买入信号数、止盈触发数、止损触发数。
- 执行统计：下单数、成交数、成交率、未成交原因分布。
- 收益统计：总收益、年化、最大回撤、费用后净收益。
- 阶梯止盈统计：每个止盈步骤的触发次数与贡献收益。
