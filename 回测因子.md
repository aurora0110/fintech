## 一、各策略的核心目的
| 策略 | 目的 |
|---|---|
| `B1` | 找低位修复、低风险回踩反弹 |

## 二、已实际用到的因子
### 1. 趋势背景因子
| 因子 | 含义 | 已用于哪些策略 |
|---|---|---|
| `趋势线 > 多空线` | 短中期结构偏强 | `B1`、`B2`、`B3`、`单针`、`BRICK` |
| `趋势线5日斜率 > 0` | 短期趋势向上 | `B2` 研究过，`单针` A/B/C 实际用，`BRICK` 排序里间接用 |
| `趋势线10/20/30日斜率 > 0` | 中短期趋势向上 | `B1` 回测过，`单针` 研究过 |
| `多空线5日斜率 > 0` | 中期结构抬升 | `B2`、`BRICK` 研究里有意义 |
| `趋势线领先多空线幅度` | 趋势强度/扩张程度 | `单针` A/B/C、`B2` 研究中用过 |

### 2. 低位/位置因子
| 因子 | 含义 | 已用于哪些策略 |
|---|---|---|
| `J < 13` | 极低位超卖 | `B1` 历史版、`B3` 前置链条当前版 |
| `J < 20日历史10%分位` | 低位但更自适应 | `B1` 当前版、`B2第一类` |
| `收盘在趋势线上/下、在多空线上/下` | 所处位置 | `B1`、`B2`、`B3`、`BRICK`、周线版 `B1/B3/单针` |
| `相对前高位置` | 是中高位活动区还是追突破/低位反抽 | `BRICK` 排序与研究中已用 |
| `signal_vs_ma20 / dist_trend / dist_long` | 与均线/趋势线偏离程度 | `单针`、`B2`、`BRICK` 研究中用过 |

### 3. 涨幅/节奏因子
| 因子 | 含义 | 已用于哪些策略 |
|---|---|---|
| `ret1 >= 4% / 3%` | 启动阳线力度 | `B2` |
| `当日涨幅 < 2%` | 缩量承接而不是继续追涨 | `B3` |
| `当日振幅 < 5%` | 波动温和 | `B3` |
| `ret3 <= 1%` | 近期温和回踩 | `单针` |
| `ret5/ret10` | 前期是否已有涨幅基础 | `单针`、风险研究 |
| `前5日回调 <= -1%` | 前面先压一压 | `BRICK` 当前硬条件之一 |

### 4. 成交量因子
| 因子 | 含义 | 已用于哪些策略 |
|---|---|---|
| `缩量` | 风险降低、承接更稳 | `B1`、`B3`、`单针A/C` |
| `B2日比B1日放量` | 启动得到确认 | `B3` 前置基础B2链条 |
| `signal_vs_ma5 / ma10 / ma20` | 当日量能相对均量 | `单针`、`BRICK`、`B2` |
| `vol_vs_prev` | 相对前一日放量/缩量 | `B3`、`单针`、`BRICK` |
| `当日量 >= 前一日2倍` | 强确认量 | `BRICK` 额外标注/加分 |
| `阶段性放量` | 真正有资金发动 | `B1` 历史因子里偏强正贡献 |

### 5. K线质量因子
| 因子 | 含义 | 已用于哪些策略 |
|---|---|---|
| `上影线 < 实体 1/3` | 启动不虚、不被打回 | `B3` 前置基础B2 |
| `上影线/实体比例` | 启动质量 | `B2` 成功案例统计里重点研究过 |
| `下影线占比` | 洗盘/探底质量 | `单针` |
| `收盘位置 close_position` | 收得扎不扎实 | `单针`、`BRICK`、风险标签 |
| `阳线/阴线` | 基础方向 | `B2`、`B3`、`BRICK`、`B1` 某些触发项 |
| `实体长度/振幅(body_ratio)` | K线实体是否有效 | `B2`、`BRICK`、风险标签 |
| `红砖长度 > 前一日绿砖 * 0.66` | 反包最低门槛 | `BRICK` |

## 三、已经验证“可能有用”的因子
这些不是全部都已正式写死，但历史研究里已经证明有信息量。

### 1. 风险因子
| 因子 | 含义 | 更适合哪个策略 |
|---|---|---|
| `近20日高位巨量阴线` | 前期已经出现明显出货 | `单针` 最有用，`B2` 次之 |
| `近20日假突破` | 冲高失败、上方抛压重 | `BRICK`、`B2` |
| `近20日顶部区间出货` | 顶部阴量主导 | `B3` 最明显 |
| `连续阳线阶段斜率过陡` | 前期拉升太快 | `BRICK`、部分 `B2` |
| `近20日阶梯量` | 前期结构已脏 | `BRICK`、`单针`、`B2` 可研究，但定义还需继续收紧 |

### 2. 结构增强因子
| 因子 | 含义 | 更适合哪个策略 |
|---|---|---|
| `N型向上` | 有中继/再起结构 | `单针C`、`BRICK`、`B3` |
| `沿趋势线上涨` | 顺主趋势活动 | `单针C`、`BRICK` |
| `关键K支撑` | 结构支撑明确 | `单针C`、`BRICK` |
| `反包比` | 当前红K/红砖对前一根绿K/绿砖的扭转力度 | `BRICK` 最重要，`B2` 次之 |
| `连续绿砖数量` | 压制后转强的节奏 | `BRICK` |
| `中高位活动区` | 不是深跌反抽，也不是纯突破追价 | `BRICK`、`B2` |

### 3. 候选排序因子
| 因子 | 含义 | 更适合哪个策略 |
|---|---|---|
| `当日量是前一日2倍` | 明显确认量 | `BRICK`、`B2` |
| `前5日3阴` | 前期先压过 | `BRICK`、`单针A` |
| `红砖/红K在近20天里偏大` | 当前这根确实有力度 | `BRICK`、`B2` |
| `上影线更短` | 收得扎实 | `B2`、`B3`、`BRICK` |
| `缩量更明显` | 承接更稳 | `B3`、`单针A/C` |

## 四、每个策略最适合用哪些因子
### B1
更适合：
- `趋势线 > 多空线`
- `J低位`
- `回踩均线 / 回踩趋势线 / 第一次回踩 / 地量低价 / 长阴短柱`
- 周线强势背景

谨慎使用：
- 复杂加速/出货风险硬过滤  
原因：B1 本身就是偏低位修复，很多“透支”标签不一定适用。

### B2
更适合：
- `趋势线 > 多空线`
- `启动涨幅`
- `量比/放量`
- `上影线短`
- `靠近多空线低位`
- `前一日是B1或低位结构`
- `假突破/高位巨阴` 风险过滤

### B3
更适合：
- `前一日基础B2`
- `前前日 J<13 且前一日J更高`
- `当日小涨`
- `当日小振幅`
- `缩量`
- 周线强势背景
- `顶部出货区` 风险过滤

### 单针
更适合：
- `趋势线 > 多空线`
- `ret3/ret10`
- `vol_vs_prev`
- `signal_vs_ma20`
- `下影线占比`
- `A/B/C 结构`
- `N型 / 沿趋势 / 关键K支撑`
- `高位巨量阴线` 风险过滤

### BRICK
更适合：
- `砖色节奏`
- `红砖长度 / 反包比`
- `趋势线 > 多空线`
- `收盘站上趋势线`
- `量比五日`
- `前5日回调`
- `前5日3阴`
- `中高位活动区`
- `2倍量`
- `假突破 / 顶部出货 / 连续阳线斜率过陡` 风险过滤或扣分

## 六、一句话总结
如果压缩成一句话：

- **B1** 主要吃“低位 + 修复”
- **B2** 主要吃“启动质量”
- **B3** 主要吃“启动后的承接确认”
- **单针** 主要吃“强趋势里的洗盘后再起”
- **BRICK** 主要吃“连续压制后的强反包”

而跨策略最有价值的公共因子，就是：
- **趋势背景**
- **低位程度**
- **量价确认**
- **K线质量**
- **结构标签**
- **风险标签**

如果你要，我下一步可以把这份整理继续压成一张更清晰的总表，例如：

- 行：因子
- 列：`B1 / B2 / B3 / 单针 / BRICK`
- 单元格写：`已用 / 可用 / 不适合`

这样你后面一眼就能看出每个因子该放到哪个策略里。
